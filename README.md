# aoc-polyglot

Solutions to a few Advent of Code puzzles in different languages. All run instructions are for Windows, but the code should run on any system. 

## 2015 day 1: C#

```
dotnet run
```

## 2015 day 2: Python

```
python aoc.py
```

## 2015 day 3: Javascript

```
node aoc.js
```

## 2015 day 4: Go

```
go run aoc.go
```

## 2015 day 5: Ruby

```
ruby aoc.rb
```

## 2015 day 6: Typescript

```
npx ts-node aoc.ts
```

## 2015 day 7: Java

```
javac aoc.java;java aoc
```

## 2015 day 8: Clojure

```
lein run
```

## 2015 day 9: Visual Basic

```
dotnet run
```

## 2015 day 10: F#

```
dotnet run
```

## 2015 day 11: Kotlin

```
kotlinc-jvm aoc.kt;kotlin AocKt
```

## 2015 day 12: Rust

```
cargo run
```

## 2015 day 13: Erlang

```
erl -compile aoc;erl -noshell -s aoc run -s init stop
```

## 2015 day 14: Groovy

```
groovy .\aoc.groovy
```

## 2015 day 15: Scala

```
sbt run
```

## 2015 day 16: Haskell

```
ghc aoc.hs;./aoc.exe
```

## 2015 day 17: Dart

```
dart run
```

## 2015 day 18: C++

```
cl /EHsc aoc.cpp
aoc.exe
```

## 2015 day 19: Julia

```
julia aoc.jl
```

## 2015 day 20: Haxe

```
haxe --main Aoc --interp
```
