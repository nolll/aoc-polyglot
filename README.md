# aoc-polyglot

Solutions to a few Advent of Code puzzles in different languages

## c#

```
dotnet run aoc
```

## go

```
go run 2015-01.go
```

## javascript

```
node 2015-01.js
```

## python

```
python 2015-01.py
```

## ruby

```
ruby 2015-01.rb
```

## typescript

```
npx ts-node 2015-01.ts
```
