# aoc-polyglot

Solutions to a few Advent of Code puzzles in different languages

## 2015 day 1: C#

```
dotnet run aoc
```

## 2015 day 2: Python

```
python aoc.py
```

## 2015 day 3: Javascript

```
node aoc.js
```

## 2015 day 4: Go

```
go run aoc.go
```

## 2015 day 5: Ruby

```
ruby aoc.rb
```

## 2015 day 6: Typescript

```
npx ts-node aoc.ts
```

## 2015 day 7: Java

```
javac aoc.java;java aoc
```

## 2015 day 8: Clojure

```
lein run
```
