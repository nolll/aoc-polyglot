# aoc-polyglot

Solutions to a few Advent of Code puzzles in different languages

## 2015 day 1: C#

```
dotnet run aoc
```

## 2015 day 2: Python

```
python aoc.py
```

## 2015 day 3: Javascript

```
node aoc.js
```

## 2015 day 4: Go

```
go run aoc.go
```

## ruby

```
ruby 2015-01.rb
```

## typescript

```
npx ts-node 2015-01.ts
```
